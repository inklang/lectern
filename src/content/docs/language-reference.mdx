# Language Reference

## Variables

```ink
let x = 10
const PI = 3.14
```

## Functions

```ink
fn greet(name) {
  return "Hello, " + name
}
```

## Control Flow

```ink
if x > 10 {
  print("big")
} else {
  print("small")
}
```

## Loops

```ink
for i in range(10) {
  print(i)
}
```
